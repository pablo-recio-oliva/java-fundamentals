# Inventario de biblioteca — Java

Proyecto de consola para gestionar un inventario básico de libros y préstamos. Está pensado como ejercicio de repaso de Java usando únicamente clases sencillas, objetos, listas, bucles y validaciones básicas.

## 📌 Descripción

La aplicación permite registrar libros, mostrar el catálogo, buscar por título, prestar libros y devolverlos.

## 🧠 Conceptos trabajados

- Clases y objetos
- Constructores
- Encapsulación
- `ArrayList`
- Bucles `for` y `while`
- Condicionales
- Métodos auxiliares
- Entrada de datos con `Scanner`

## ▶️ Ejecución

```bash
cd 02-inventario-biblioteca
javac -d out src/*.java
java -cp out Principal
```


## Ejecución

Este proyecto puede abrirse directamente desde NetBeans como proyecto Java. La clase principal es `Principal.java`.
