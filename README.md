Colección de proyectos de **Java** desarrollados y organizados como parte de la formación de Desarrollo de Aplicaciones Multiplataforma (DAM). El repositorio está orientado a repasar conceptos fundamentales de programación: clases, objetos, bucles, condicionales, arrays, listas, herencia, interfaces y menús por consola.

## 📁 Estructura del repositorio

| Carpeta | Contenido |
|---|---|
| [`01-operativo-ada/`](./01-operativo-ada) | Proyecto original de repaso general de Java con POO, herencia, interfaces, colecciones y simulación por consola |
| [`02-inventario-biblioteca/`](./02-inventario-biblioteca) | Inventario de biblioteca con libros, préstamos, búsquedas y gestión por menú |
| [`03-gestor-alumnos/`](./03-gestor-alumnos) | Gestor de alumnos con notas, medias, expedientes y clasificación de aprobados/suspensos |

Cada carpeta tiene su propio README con el detalle del proyecto y qué conceptos demuestra.

## 🚀 Cómo ejecutarlo

Cada proyecto se puede compilar y ejecutar de forma independiente. Por ejemplo:

```bash
cd 02-inventario-biblioteca
javac -d out src/*.java
java -cp out Principal
```

Para `01-operativo-ada`, al conservar la estructura original de NetBeans, también se puede abrir directamente desde ese IDE.

## 🎯 Qué se ha cuidado especialmente

- **Código sencillo de seguir**: proyectos de consola sin estructuras complejas innecesarias.
- **Programación orientada a objetos**: uso de clases, atributos, métodos, constructores y encapsulación.
- **Lógica práctica**: menús, búsquedas, validaciones y cálculos.
- **Organización**: cada proyecto está separado en su propia carpeta.
- **Documentación**: README individual para explicar el objetivo de cada ejercicio.
