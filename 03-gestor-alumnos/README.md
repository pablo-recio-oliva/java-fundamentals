# Gestor de alumnos — Java

Proyecto de consola para gestionar alumnos, asignaturas y calificaciones. Es un ejercicio orientado a practicar objetos, listas, lógica de cálculo y menús interactivos.

## 📌 Descripción

La aplicación permite registrar alumnos, añadir notas, calcular medias, mostrar aprobados/suspensos y consultar el expediente de un alumno.

## 🧠 Conceptos trabajados

- Clases y objetos
- `ArrayList`
- Bucles y condicionales
- Métodos de búsqueda
- Cálculo de medias
- Validación de entrada
- Separación sencilla entre el `Principal` y las clases del programa
